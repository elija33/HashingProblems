# HashingProblems

Given an array which may contain duplicates, print all elements and their frequencies.

Examples:

Input :  arr[] = {10, 20, 20, 10, 10, 20, 5, 20}
Output : 10 3
         20 4
         5  1

Input: arr[] = {10, 12, 10, 15, 10, 20, 12, 12}
Output: 10 3
        12 3
        15 1
        20 1

Input: arr[] = {10, 10, 10, 10}
Output: 10 4

Input : arr[] = {10, 20, 20}
Output : 10 1
         20 2
